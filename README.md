# HyperConfig

Configuration file for [Hyper](https://hyper.is/) 

* Download and Install

* Configure
Copy `.hyper.js` to configuration directory.

| OS      | location                            |
| ------- | ----------------------------------- |
| windows | %USERPROFILE%\AppData\Roaming\Hyper |
| linux   | ~/.config/Hyper/.hyper.js           |
